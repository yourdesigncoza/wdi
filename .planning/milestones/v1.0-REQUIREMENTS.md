# Requirements Archive: v1.0 MVP

**Archived:** 2026-02-08
**Status:** SHIPPED

**Defined:** 2026-02-05
**Core Value:** Any South African can create a legally compliant will through an intelligent, guided conversation — no legal knowledge required.

## v1 Requirements

### Authentication

- [x] **AUTH-01**: User can register with email and password — Phase 2
- [x] **AUTH-02**: User receives email verification after registration — Phase 2
- [x] **AUTH-03**: User can log in and stay logged in across sessions — Phase 2
- [x] **AUTH-04**: User can reset password via email link — Phase 2
- [x] **AUTH-05**: User can save progress and resume will creation later — Phase 8
- [x] **AUTH-06**: User can update their will unlimited times after purchase — Phase 8

### Will Creation

- [x] **WILL-01**: User can create a basic will through guided questionnaire — Phase 3
- [x] **WILL-02**: User can designate beneficiaries (individuals or charities) — Phase 3
- [x] **WILL-03**: User can inventory assets (property, vehicles, bank accounts, investments) — Phase 3
- [x] **WILL-04**: User can nominate primary and backup guardians for minor children — Phase 3
- [x] **WILL-05**: User can nominate executor(s) for estate administration — Phase 3
- [x] **WILL-06**: User can specify alternate beneficiaries (if primary predeceases) — Phase 3
- [x] **WILL-07**: User can specify specific bequests (particular items to particular people) — Phase 3
- [x] **WILL-08**: User can designate residual estate beneficiaries — Phase 3

### Document Generation

- [x] **DOC-01**: System generates professionally formatted PDF will document — Phase 6
- [x] **DOC-02**: PDF includes clear witness signing instructions — Phase 6
- [x] **DOC-03**: User can preview document before payment — Phase 6
- [x] **DOC-04**: User must complete payment (PayFast) before downloading PDF — Phase 7
- [x] **DOC-05**: User receives download link immediately after payment — Phase 7
- [x] **DOC-06**: User receives email with download link as backup — Phase 7

### Legal Compliance

- [x] **COMP-01**: User must accept POPIA consent before starting — Phase 1
- [x] **COMP-02**: Privacy policy clearly explains data handling — Phase 1
- [x] **COMP-03**: Information Officer contact details are displayed — Phase 1
- [x] **COMP-04**: User can request data access, correction, or deletion — Phase 1
- [x] **COMP-05**: Disclaimers clearly state "not legal advice" at key touchpoints — Phase 6
- [x] **COMP-06**: System recommends attorney review for complex scenarios — Phase 6

### AI Features

- [x] **AI-01**: AI guides user through will creation conversationally — Phase 3
- [x] **AI-02**: AI provides real-time explanations of legal terms and concepts — Phase 3
- [x] **AI-03**: AI handles "I don't know" responses gracefully with follow-up questions — Phase 3
- [x] **AI-04**: AI validates inputs and catches potential errors — Phase 3
- [x] **AI-05**: Secondary LLM verifies document completeness before generation — Phase 5

### Complex Scenarios

- [x] **CMPLX-01**: User can create will with blended family scenarios — Phase 4
- [x] **CMPLX-02**: User can create testamentary trust provisions — Phase 4
- [x] **CMPLX-03**: User can add usufruct provisions — Phase 4
- [x] **CMPLX-04**: User can handle business assets — Phase 4
- [x] **CMPLX-05**: User can create joint will with spouse — Phase 4

### Additional Documents

- [x] **ADOC-01**: User can create living will (advance healthcare directive) — Phase 9
- [x] **ADOC-02**: User can create funeral wishes document — Phase 9
